What this editor supports

  • Assignment with <- or =
  • Numbers, strings, TRUE/FALSE
  • Vectors via c(...) and ranges via 1:10
  • Arithmetic: + - * / on numbers and vectors
  • data.frame(col = c(...), ...) with real table rendering
  • df$column access on data frames
  • Two built-in datasets, usable by name: students, sales
  • print(), cat(), and bare-variable auto-print
  • plot(), hist(), barplot() rendered on canvas from your actual data
  • Built-ins: c, length, sum, mean, round, seq, paste, nrow, ncol

Not supported: control flow (if/for/while), custom functions, packages, or most of the R standard library. This runs a small interpreter built for this editor, not a full R engine.
